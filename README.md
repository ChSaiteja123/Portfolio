# 🚀 DevOps Portfolio

A modern, responsive, and animated **DevOps Portfolio Website** built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. The portfolio showcases expertise in **Cloud Computing, Kubernetes, Docker, CI/CD, DevSecOps, Infrastructure as Code (IaC), Monitoring, and Automation** through a futuristic cyberpunk-inspired design.

---

## ✨ Features

* 🎨 Modern cyberpunk-inspired UI
* 📱 Fully responsive design
* ⚡ Fast performance powered by Vite
* 🎥 Smooth animations with Framer Motion
* ☁️ DevOps workflow visualization
* 💼 Professional portfolio sections:

  * Hero
  * About
  * Skills
  * Experience
  * Projects
  * Education
  * Certifications
  * Contact
* 📄 Downloadable Resume
* 🌙 Dark theme
* 🔥 Interactive hover effects and transitions

# 📋 Prerequisites

Before running the project, install:

* Node.js 18+
* npm 9+

Verify installation:

```bash
node -v
npm -v
```

---

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/devops-portfolio.git
```

Navigate into the project:

```bash
cd devops-portfolio
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Run the Development Server

Start the application:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

*(Vite typically uses port 5173 by default.)*

---

# 📦 Production Build

Generate an optimized production build:

```bash
npm run build
```

The build output will be created inside:

```text
dist/
```
